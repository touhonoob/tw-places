# Taiwan Places

This is a practice project.

## Main Goals:

* Practice react.js with rails
* Familiar mongodb
* Familiar Nokogiri
* Have fun


## Setup

`bundle install`

開啟 mongodb `mongod`

`rake parse_gov_place_api`

`rails s`

go to `localhost:3000`
